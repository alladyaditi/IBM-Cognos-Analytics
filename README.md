# Customer Loyalty Program Dashboard — IBM Cognos Analytics

This repository contains my first analytics dashboard created as part of the **IBM Data Analytics Professional Certificate**. The project uses the **Customer Loyalty Program** dataset to explore customer behavior, purchasing patterns, and engagement trends through interactive visualizations built in **IBM Cognos Analytics**.

## Project Overview

The dashboard analyzes key aspects of customer loyalty, including:

- Customer demographics  
- Purchase frequency and product-line trends  
- Reward redemption behavior  
- Overall engagement metrics  

The visualizations include year-over-year product performance, total quantity sold, and revenue insights, helping illustrate how different customer segments interact with the loyalty program.

## Objectives

This project was designed to help me practice:

- Data modeling and preparation  
- Building dashboards in IBM Cognos Analytics  
- Designing clear, insight-driven visualizations  
- Using Cognos’ AI-assisted analytics features  
- Interpreting customer behavior for business decision-making  

## Tools & Technologies

- IBM Cognos Analytics
